terminal-status
===============

## Usage
Just press `shift-enter`

## Feature

* multiple terminal
* colorful status icon
* kill long live process
* fancy ls

## Hotkeys

* `shift-enter` toggle current terminal
* `command-shift-e` new terminal
* `command-shift-j` next terminal
* `command-shift-k` prev terminal
